Build Live Arch ISO

Using this repository, Archiso and some additional configurations, one can build a live Arch ISO image. To recreate what I have built, download livearch.tar.gz, extract it with tar -xvf livearch.tar.gz to obtain the livearch folder, then sudo chown -R root:root livearch, and run ./build.sh -v inside the livearch folder as root. (You may need around 25 - 30 GB of free disk space, and the package archiso.)
